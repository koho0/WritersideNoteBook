## 推送项目到github
```Bash
# 初始化本地 Git 倉庫
git init

# 添加所有檔案到暫存區
git add .

# 提交變更
git commit -m "第一次提交"

# 連接遠端倉庫
git remote add origin https://github.com/koho0/WritersideNote

# 推送變更
git push -u origin main
```
